Using the Quiz Solver AI API

Overview

The Quiz Solver AI API allows you to solve quiz questions programmatically using artificial intelligence. This page provides detailed instructions on how to use the API, including examples of HTTP requests and responses.

Endpoint

The main endpoint for using the Quiz Solver AI API is /question. This endpoint accepts POST requests containing the question to be solved and returns the correct answer.

Request Format

The request to the /question endpoint should be in JSON format with the following structure: 

{
    "question": "YOUR_QUESTION_HERE"
}

Response Format

The response from the API will be in JSON format with the following structure: 

{
    "answer": "CORRECT_ANSWER"
}

Example

Below is an example of how to use the Quiz Solver AI API to solve a quiz question:

Request:

{
    "question": "Question: What is the capital of France?\nA. London\nB. Paris\nC. Berlin\nD. Rome"
}

Response:

{
    "answer": "B"
}

Usage Guidelines

  • Ensure that the question is formatted correctly and includes the options in the specified format (A, B, C, D).
  • Handle errors gracefully by checking the response status code and message.
  • Consider implementing rate limiting to prevent abuse of the API.
